# CopyOdds 使用文档

CopyOdds 是基于 Polymarket 的聪明钱分析与自动跟单平台。本文档帮助您完成账户注册、资金准备、Gas 购买、聪明钱筛选与自动跟单设置。

## 使用流程总览

1. **注册账户** — 使用邮箱创建 CopyOdds 账户
2. **充值 USDC** — 通过 Polygon (POS) 向账户充值交易本金
3. **购买平台 Gas** — 为自动跟单服务充值点数
4. **选择聪明钱** — 在排行榜中筛选优质交易员
5. **设置跟单规则** — 配置跟单对象、方向与金额上限
6. **查看交易结果** — 在交易记录中确认执行与盈亏

## 适用环境

- Web 浏览器、移动浏览器
- PWA（可添加到主屏幕）
- 网络与资产：Polygon (POS) 上的 USDC
