# CopyOdds 使用文档

CopyOdds 是基于 Polymarket 的聪明钱分析与自动跟单平台。按下面步骤操作即可完成注册、充值、选人和跟单。

## 使用流程总览

1. **注册账户** — 用邮箱注册 CopyOdds
2. **充值本金** — 向钱包页显示的托管地址充入 Polygon 上的 USDC.e / USDC / USDT0 等
3. **购买平台 Gas** — 为自动跟单充值服务费点数
4. **选择聪明钱** — 在排行榜里挑选要跟单的交易员
5. **设置跟单规则** — 填写地址、跟单方式和金额上限
6. **查看交易记录** — 在「交易」页确认是否跟单成功

## 开始跟单前，请确认

| 项目 | 要求 |
|------|------|
| 平台 Gas | 大于 0（否则无法开启或恢复跟单） |
| USDC 余额 | 建议至少准备约 $1，用于实际买入跟单 |
| 交易账户 | 登录后通常会自动开通，无需手动操作 |

## 适用环境

- 电脑或手机浏览器均可使用
- 可添加到主屏幕（PWA）
- 充值网络：Polygon（POS）；资产：USDC.e（首选）、原生 USDC、USDT0、Wormhole USDT（勿用 BSC 等错链）
