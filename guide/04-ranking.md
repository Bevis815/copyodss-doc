# 排行榜说明

「聪明钱」排行榜帮助您发现表现较好的 Polymarket 交易员，选择合适的跟单对象。数据由系统定期更新，**不是实时数据，也不构成投资建议**。

***

## 查看聪明钱排行榜

### 操作步骤

1. 进入「**聪明钱**」页面。
2. 查看榜单汇总信息：交易员数量、平均胜率、平均评分等。
3. 切换统计周期：**全部时间**、**周榜**、**月榜**。
4. 浏览交易员卡片，查看盈利、胜率、交易次数、当前资金和风险标记。
5. 使用搜索或筛选查找目标交易员。
6. 点击「**查看详情**」，查看收益走势、最大回撤、持仓估值等。
7. 决定跟单时，点击「**去跟单**」或「跟单」。

![聪明钱排行榜](../.gitbook/assets/smarket_doc.png)

_排行榜页：切换周期并浏览交易员卡片，进入详情或发起跟单。_

## 卡片上的信息怎么读

| 信息       | 含义                                        |
| -------- | ----------------------------------------- |
| **综合评分** | 系统根据收益质量、活跃度、稳定性等维度给出的综合分数，分数越高通常代表综合表现越好 |
| **胜率**   | 历史预测中猜对的比例；胜率高不代表一定赚钱，还要看交易次数             |
| **收益**   | 历史盈利情况；过去赚过不代表未来也会赚                       |
| **交易次数** | 样本太少时，胜率和收益参考价值有限                         |
| **风险标记** | 系统标注的异常信号，如收益波动过大、近期表现下滑等，请留意             |
| **持仓估值** | 该交易员当前持仓的大致价值                             |

### 关于风险标记

排行榜可能显示风险标记，表示该交易员在某些维度触发了系统的风险提示。常见含义：

* 收益曲线波动剧烈
* 近期表现明显下滑
* 持仓规模偏低
* 交易样本较少

**没有风险标记，不代表没有风险**；预测市场本身就有波动，请结合自身情况判断。

***

## 使用建议

* 不要只看单一指标就决定跟单，建议综合评分、胜率、交易次数和风险标记一起看。
* 新用户建议先**小金额测试**，熟悉充值、Gas、跟单和交易记录后再逐步调整。
* 排行榜数据仅供参考，跟单前请评估自己能承受的风险。

### 注意事项

* 排行榜数据定期更新，与链上最新成交存在延迟。
* 收益数据仅供参考，过去表现不代表未来收益。
* 没有排名数字的交易员，表示当前未进入榜单前列，不代表一定不好。
